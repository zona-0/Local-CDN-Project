What is TCP/IP?

TCP/IP is the communication model used on the Internet. It consists of 4 layers that manage how data is sent and received.

How Data Travels (Step-by-Step)

1️⃣ Application → User sends request (HTTP)
2️⃣ Transport → TCP adds reliability & port
3️⃣ Internet → IP routes packet
4️⃣ Network → Data sent via cable/Wi-Fi

TCP/IP vs OSI

TCP/IP

  • 4 Layers
  • Used in real networks
  • Practical model

OSI

  • 7 Layers
  • Theoretical model
  • Learning framework

Mapping to OSI

TCP/IP layers correspond to OSI layers:

  • Application → OSI Layer 5–7
  • Transport → OSI Layer 4
  • Internet → OSI Layer 3
  • Network Access → OSI Layer 1–2
